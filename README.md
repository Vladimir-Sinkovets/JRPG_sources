This repository contains the custom C# scripts for a Unity RPG prototype. It includes core mechanics like inventory, equipment, turn‑based combat, AI behaviors, and save/load systems. This is a learning project to practice game architecture, design patterns, and working with Unity.

> The full project uses some paid assets from the Unity Asset Store, so I can’t include them here. This repository only has the scripts I wrote – no art, scenes, or third‑party files. You can browse the code to see my implementation style and structure.
